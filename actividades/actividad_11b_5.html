CSS Selector

Hay muchos tipos diferentes de selector CSS que le permite orientar las reglas a eelementos especificos en un documento HTML.

Los selectores CSS distinguen entre mayusculas y minusculas, por lo que deben coincidir exactamente con los nombres del elemento y los valores del atributo.

Tipos de Selectores
Selector Significado Ejemplo
Universal Selector Coincide con los nombres del elemento * {} Se dirige a todoos los elementos de la pagina
Type Selector Coincide con los nombres del elemento h1, h2, h3 {} Corresponde a los elementos <h1>, <h2> y <h3>
Class Selector Coincide con un elemento cuyo atributo de clase tiene un valor que corresponda con el especificado despues del simbolo del punto .nota {} Se dirige a cualquier elemento cuyo atributo de clase tenga un valor de nota
Id Selector Coincide con un elemento cuyo atributo de clase tiene un valor que corresponda con el especificado despues del simbolo del hast #introduction {}
