Adding HTML5 Video

Un video de un cachorro jugando en la nieve

En HTML5 no es necesario proporcionar vaores para todos los atributos, como los controles, reproduccion automatica y atributos de blucle utilizandos con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo esta presente, activa esa opcion. Si se omite el atributo, la opcion esta desactivada, si el navegador no admite el emneto <video> de apertura y las etiquetas de </video> de cierre;

preload: Este atributo le indica al navegador que hacer cuando se carga la pagina. Puede tener uno de tres valores:
none: El navegador no debe cargar el video hasta que el usario presiona play.
auto: El navegador debe descargar el video cuando se carga la pagina,
metadata: El navegaro debe simplemente recopilar informacion como el tama√±o, el primer fotograma, la lista de pistas y la duracion.

Multiple Video Sources

Un video de un cachorro jugando en la nieve

La etiqueta video: Para especificar la ubicacion del archivo que se reproducira, puede usar el elemento <source> dentro de elemento <video>, (Esto dee reemplezar el atributo src en la etiqueta <video> de apertura.) Tambien puede utilizar varios elementos <source> para especificar que el esta disponible en diferentes formatos. (Debido a un error el ipad, debe proporcionar el video MP4 como el primer formato. De lo contrario, puede que no inicie.)

src: Este atributo especifica la runa de acceso al video

type: Se debe ulitizar este atributo para decirle al navagador que formato es el video. De lo contrario, descargara parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda)

codecs: El codec que uso para codificar el video se suministra dentro del atributo type. Tenga en cuenta el uso de comillas simples, asi como comillas dobles en el atributo de type, cuando se suministra

Adding HTML5 Audio

<audio< HTML5 introdujo el elemento <audio> para incluir archivos de audio en sus paginas, Al igual que con el video HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un numero de atributos que le permiten controlar la reproduccion de audio:

src: Este atributo especifica la runta de acceso al archivo de audio

controls: Este atributo indica si el reproductor debe mosntrar los controles. Si no utiliza este atributo, no se mostraran los contrloes de forma predeterminada. Tmabien puede especificar sus propios controles mendiante javascript

autoplay: La precensia de este atributo indica que el audio debe empezar a reproducirse automatiamente. (Se comside una mejor practica permitir al usario que reprodusca el audio)

loop: Este atributo especifica que las pista de audio debe reproducirse de nuevo una ves que haya finalizado

HTML Youtube Videos

Youtube video ID, youtube mostrara un ID (Como EF81pb5IWou), al guardar (o reproducir) un video. Puede utilizar este ID y hacer referencia a su video en el codigo HTML.

Reproducion de un video de youtube en HTML:

  • Sube el vidio a youtube
  • Anota el ID del vidio
  • Defina el elemento <iframe> en su pagina web
  • Deje que el atributo src apunte a la URL del video
  • Utilice los atributos widht y height para especificar la dimension del reproductor
