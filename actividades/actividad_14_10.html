La propiedad box-shadow permite agregar una sombra paralela alrededor de un caja. Funciona igual que la propiedad text-shadow. Chrome, Firefox y Safari admiten es propiedad con las propiedades: -moz-box-shadow y -webkit-box-shadow. Estos no estan es la especificacion CSS pero usar los pueden ayudar a este estilo para trabajar n estos navegadores.
